# Proyecto 4: SQL

## Descripción
Este repositorio contiene la resolución del **Proyecto 4: SQL**, cuyo objetivo es poner en práctica consultas de lectura sobre una base de datos PostgreSQL.

El proyecto se ha realizado utilizando la base de datos de ejemplo **Demo Bookings** proporcionada por PostgresPro, trabajando principalmente sobre el esquema `bookings`.

---

## Base de datos utilizada
- Nombre de la base de datos: `demo`
- Esquema principal: `bookings`
- Origen de la base de datos:
  https://postgrespro.com/docs/postgrespro/current/demodb-bookings-installation.html
- Versión utilizada: **demo-small-en**

---

## Tablas utilizadas
Durante la práctica se han utilizado, entre otras, las siguientes tablas:
- `flights`
- `bookings`
- `tickets`
- `ticket_flights`
- `airports_data`
- `aircrafts_data`
- `boarding_passes`

  - Ejercicios opcionales (6 al 9)

## Resumen del trabajo realizado
1. Descarga de la base de datos Demo Bookings (versión ligera).
2. Creación de la base de datos `demo` en PostgreSQL.
3. Importación del archivo SQL utilizando **SQL Shell (psql)**, ya que el fichero incluye comandos específicos de psql.
4. Verificación de la correcta carga del esquema `bookings` y sus tablas.
5. Ejecución y comprobación de cada consulta SQL en pgAdmin.
6. Copia final de las consultas en el archivo `IntroBBDD.sql`.
